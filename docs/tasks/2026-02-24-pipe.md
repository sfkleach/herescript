# Pipe the header block into a shell

Annoyingly I have realised that there is a very elegant mechanism that I
missed the first time around. The idea is that the header block is the set of 
lines that start `#:`, skipping the initial `#!`. We then treat that text,
with the `#:` prefixed removed, as a script to be executed. 

## Step 1: Support `#:`

Support `#:` as another prefix for header lines. Collect it, eliding the prefix.
If the collected string is non-empty, run "/bin/sh -c HEADER_SCRIPT" with the
environment variable $SCRIPT_FILE defined as the absolute name of the file.

## Step 2: Support no executable

When herescript is invoked with no extra argument, this is a request to only 
use the /bin/sh invocation.
